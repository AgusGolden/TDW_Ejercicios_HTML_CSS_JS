Ejercicios de HTML CSS y JAVASCRIPT

  • Ejercicio 1

    To-Do List interactiva: Diseña una lista de tareas donde puedas agregar, tachar y eliminar tareas. Implementa la interacción con JavaScript, el diseño visual con CSS y la organización con HTML.

  • Ejercicio 2

    Formulario de contacto validado: Haz un formulario que pida nombre, correo y mensaje. Agrega validación de campos usando JavaScript y personaliza el estilo del formulario con CSS.

  • Ejercicio 3

    Galería de imágenes con efectos: Muestra varias imágenes en una cuadrícula. Al hacer clic en una imagen, que se muestre ampliada (modal). Aplica efectos de transición con CSS y controla el modal con JavaScript.

  • Ejercicio 4

    Reloj digital en tiempo real: Crea un reloj que muestre la hora actual y se actualice cada segundo usando JavaScript.

  • Ejercicio 5

    Menú de navegación responsive: Diseña un menú que cambie de formato en dispositivos móviles (hamburguesa). Usa HTML para la estructura, CSS para el diseño adaptable y JavaScript para mostrar/ocultar el menú.

  • Ejercicio 6

    Convertidor de monedas: Realiza una interfaz donde se pueda convertir entre diferentes monedas usando tasas de cambio fijas. Aplica validaciones y formato con JavaScript y CSS.

  • Ejercicio 7

    Semáforo animado: Simula un semáforo que cambia de color cada ciertos segundos. Utiliza animaciones de CSS y temporizadores de JavaScript.

  • Ejercicio 8

    Galería de testimonios con carrusel: Presenta varios testimonios en una caja que puede navegarse con flechas. Usa JavaScript para cambiar el contenido y CSS para los efectos de transición.

  • Ejercicio 9

    Gestor de contactos: Crea una aplicación donde puedas agregar, ver, editar y eliminar contactos. Los contactos deben almacenarse en un array de objetos, cada uno con propiedades como nombre, teléfono y correo. Usa HTML para el formulario y la lista, CSS para el diseño, y JavaScript para gestionar la lógica.

  • Ejercicio 10

    Inventario de productos interactivo: Desarrolla una tabla de productos donde cada producto sea un objeto en un array (propiedades: nombre, precio, cantidad, imagen). Permite añadir, eliminar y actualizar productos. Utiliza JavaScript para manipular el array y actualizar la interfaz.

  • Ejercicio 11

    Cuestionario dinámico: Implementa un quiz donde las preguntas, opciones y respuestas correctas se almacenen en un array de objetos. Muestra una pregunta a la vez, permite seleccionar respuesta y al final muestra el puntaje.

  • Ejercicio 12

    Agenda de tareas con categorías: Amplía la To-Do List usando un array de objetos, donde cada tarea tiene nombre, estado (pendiente o hecha), y una categoría (ejemplo: trabajo, estudio, personal). Permite filtrar por categoría y cambiar el estado de las tareas.

  • Ejercicio 13

    Galería de imágenes con etiquetas: Muestra una galería donde cada imagen es un objeto con propiedades como url, título y un array de etiquetas. Permite filtrar las imágenes por etiquetas seleccionando un botón.

  • Ejercicio 14

    Simulador de carrito de compras: Crea una lista de productos (array de objetos) y un carrito (array aparte). Permite agregar productos al carrito, ver el total y eliminar productos.

  • Ejercicio 15

    Semáforo configurable: Usa un objeto para definir los tiempos de cada luz y un array para el orden de los colores. Permite al usuario modificar los tiempos de cada luz y observa el cambio en la animación.

  • Ejercicio 16

    Gestor de notas: Desarrolla una aplicación donde cada nota es un objeto con título, contenido y fecha de creación. Permite ordenar por fecha, buscar por palabra clave y eliminar notas.
